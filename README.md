# vue-router-example

## 主要研究了Vue-Router的下面几点内容
- router-view, router-link指令
- Route, Router对象
- Route的定义方式，熟悉各个参数的使用方法
- 路由跳转方式
  - router-link跳转
  - $router.push()跳转
- 路由传参方式
  - 跟随router-link的to属性值传递
  - 指定$router.push()的参数传递
- 路由守卫
